## 👩‍💻 About Me

Saya merupakan peserta **Program Simulasi Industri Data Analyst** yang sedang menjalani kegiatan Praktik Kerja Lapangan (PKL) pada bidang data dan analitik.

Saya memiliki minat dalam mengolah data mentah menjadi informasi yang bernilai untuk mendukung proses pengambilan keputusan. Melalui proses pembelajaran dan pengalaman praktik, saya berfokus untuk mengembangkan kemampuan analisis data secara **end-to-end** — mulai dari pengumpulan data, pembersihan data, eksplorasi data, visualisasi, hingga penyampaian insight kepada audiens non-teknis.
